# 第四部分：马恩哲学的诞生——实践哲学的创立

> **历史背景**：19世纪40-80年代，欧洲工业革命深入发展，无产阶级形成并开始政治斗争。马克思、恩格斯在批判德国古典哲学的基础上，创立实践哲学和历史唯物主义。

## 词条16：实践哲学的创立——哲学的根本变革

### 官方解释
马克思、恩格斯在19世纪40年代创立实践哲学（praxis philosophy），核心观点：哲学的目的不是解释世界，而是改变世界。强调实践（物质生产活动）是认识的基础和检验标准，社会存在决定社会意识。

### 1840年兔狲老师解释
（我和恩格斯在伦敦的公寓里讨论手稿）

"我们不是在创立'新哲学'，而是在**终结旧哲学**——终结那种脱离实践的思辨哲学。"

**实践哲学的核心命题**（《关于费尔巴哈的提纲》，1845）：

1. **实践是认识的基础**：
   - "人的思维是否具有客观的真理性，这不是一个理论的问题，而是一个**实践**的问题。"
   - 认识不是静观，而是在**改造世界**的活动中产生的

2. **实践是检验真理的标准**：
   - "人应该在实践中证明自己思维的真理性，即自己思维的现实性和力量，自己思维的此岸性。"
   - 理论的真理性要在**实践**中检验，不是逻辑自洽

3. **实践改变社会**：
   - "环境的改变和人的活动或自我改变的一致，只能被看作是并合理地理解为**革命的实践**。"
   - 人在改造环境的同时改造自己

4. **实践的社会性**：
   - "人的本质不是单个人所固有的抽象物，在其现实性上，它是一切**社会关系**的总和。"
   - 人是在**社会实践中**形成和发展的

5. **哲学的实践转向**：
   - "哲学家们只是用不同的方式**解释**世界，问题在于**改变**世界。"
   - 哲学要从解释工具变成**改变世界的武器**

**哲学革命的意义**：
1. **终结思辨哲学**：哲学不再是无根的思辨游戏
2. **建立实践基础**：哲学扎根于**物质生产实践**
3. **改变哲学功能**：哲学成为**批判和改变现实**的理论武器
4. **统一理论和实践**：在**革命实践**中统一认识和行动

小小猪问："这和传统哲学有什么根本不同？"
我回答："传统哲学假设**思想**是第一性的，从思想推导世界。我们主张**实践**是第一性的，思想是实践的产物并反作用于实践。"

- 经验主义：通过感官经验
- 理性主义：通过理性推理

## 词条17：历史唯物主义——社会历史的新科学

### 官方解释
历史唯物主义（historical materialism）是马克思、恩格斯创立的社会历史理论，核心观点：社会存在决定社会意识，物质生活的生产方式制约着整个社会生活、政治生活和精神生活的过程。

### 1840年兔狲老师解释
（我指着工厂的机器和工人）

"要理解思想，先要理解**生产思想的社会条件**。"

**历史唯物主义基本原理**：

1. **社会存在决定社会意识**：
   - 不是意识决定生活，而是**生活决定意识**
   - 思想、观念、意识形态是**现实生活过程**的反映

2. **生产力与生产关系**：
   - **生产力**：人改造自然的能力（工具、技术、劳动力）
   - **生产关系**：人在生产中结成的社会关系（所有制、分工、分配）
   - 关系：生产力决定生产关系，生产关系反作用于生产力

3. **经济基础与上层建筑**：
   - **经济基础**：生产关系的总和
   - **上层建筑**：政治、法律、意识形态等
   - 关系：经济基础决定上层建筑，上层建筑反作用于经济基础

4. **社会形态更替**：
   - 原始社会→奴隶社会→封建社会→资本主义社会→共产主义社会
   - 动力：**生产力与生产关系的矛盾**
   - 形式：**阶级斗争**是阶级社会发展的直接动力

5. **历史发展规律**：
   - 历史不是偶然事件的堆积，而是有规律的过程
   - 规律不是机械的，而是在人的**实践活动**中实现的
   - 人创造历史，但**在既定条件下**创造

**方法论意义**：
1. **历史具体性原则**：思想要在具体历史条件下理解
2. **阶级分析方法**：揭示思想的社会基础和阶级立场
3. **总体性方法**：社会是**有机整体**，各部分相互关联
4. **实践批判方法**：不仅批判思想，而且批判产生思想的**社会条件**

小海豹问："这和黑格尔的历史哲学有什么不同？"
我回答："黑格尔认为历史是**绝对精神**的自我实现，是**思想**的历史。我们认为历史是**物质生产**的发展史，是**现实的人**在既定条件下创造的历史。"

## 词条18：放下哲学——转向政治经济学批判

### 官方解释
马克思在19世纪40年代后期逐渐放下纯粹的哲学研究，转向政治经济学批判。他发现：哲学问题的根源在于**现实的社会经济矛盾**，要批判哲学必须批判产生哲学问题的**资本主义经济关系**。

### 1840年兔狲老师解释
（大英博物馆阅览室，我面前堆满了经济资料）

"哲学批判到头了。要真正理解世界，必须研究**政治经济学**。"

**转向的原因**：

1. **发现形而上学的漏洞**：
   - 在哲学领域内，问题无法彻底解决
   - 形而上学的矛盾根源于**现实生活的矛盾**
   - 例子：自由意志问题→资本主义下的**自由**和**必然**的矛盾

2. **接触现实问题**：
   - 《莱茵报》时期：林木盗窃法、摩泽尔农民贫困
   - 发现：法律、道德问题背后是**经济利益**问题
   - 结论：要理解上层建筑，必须研究**经济基础**

3. **费尔巴哈的局限**：
   - 费尔巴哈恢复了唯物主义，但是**直观的**唯物主义
   - 只看到**自然的人**，没看到**社会历史的人**
   - 只批判**宗教异化**，没批判**经济异化**

4. **实践的需要**：
   - 无产阶级斗争需要**科学的理论武器**
   - 纯粹哲学不能指导革命实践
   - 需要研究资本主义的**经济规律**

**转向的成果**：

1. **《1844年经济学哲学手稿》**：
   - **异化劳动理论**：工人在资本主义下的四种异化
     - 与劳动产品异化
     - 与劳动过程异化  
     - 与人的类本质异化
     - 人与人相异化

2. **《德意志意识形态》**（1845-1846）：
   - 系统阐述历史唯物主义
   - 批判青年黑格尔派的"纯粹批判"

3. **《哲学的贫困》**（1847）：
   - 批判蒲鲁东的小资产阶级社会主义
   - 进一步阐发历史唯物主义

4. **《共产党宣言》**（1848）：
   - "至今一切社会的历史都是阶级斗争的历史"
   - 无产阶级的历史使命

5. **《资本论》**（1867）：
   - 政治经济学批判的顶峰
   - **剩余价值理论**揭示资本主义剥削秘密
   - **资本积累规律**揭示资本主义发展趋势

小小猪问："为什么研究经济学就能解决哲学问题？"
我回答："因为哲学问题不是**纯粹思想**的问题，而是**现实生活**在思想中的反映。现实生活的核心是**经济关系**。研究经济学就是研究现实生活的根本结构。"

## 词条19：马恩哲学的方法论——批判与革命

### 官方解释
马克思、恩格斯哲学的方法论特征：批判性、革命性、实践性、历史性、阶级性。强调理论批判与实践批判的统一，哲学批判与政治经济学批判的统一。

### 1840年兔狲老师解释
（我和恩格斯在起草《共产党宣言》）

"我们的方法不是书斋里的思辨，而是**批判的武器**和**武器的批判**的统一。"

**方法论原则**：

1. **批判性原则**：
   - 对一切现存事物进行**无情的批判**
   - 批判要**彻底**：抓住事物的根本
   - 名言："所谓彻底，就是抓住事物的根本。而人的根本就是人本身。"

2. **革命性原则**：
   - 哲学要成为**革命的理论**
   - 理论要掌握群众，变成**物质力量**
   - "批判的武器当然不能代替武器的批判，物质力量只能用物质力量来摧毁"

3. **实践性原则**：
   - 理论从**实践**中来，到**实践**中去
   - 在实践中检验和发展理论
   - 哲学要指导**改变世界**的实践

4. **历史性原则**：
   - 一切事物都在**历史过程中**理解
   - 反对永恒化、绝对化
   - 揭示思想的**历史条件**和**历史局限**

5. **阶级性原则**：
   - 思想有**阶级基础**，反映阶级利益
   - 哲学要成为**无产阶级**的精神武器
   - 从无产阶级立场出发批判旧世界

6. **总体性原则**：
   - 社会是**有机整体**，各部分相互关联
   - 要从**总体**上把握社会
   - 经济、政治、思想要统一分析

**批判的层次**：
1. **意识形态批判**：揭示思想的社会基础和阶级功能
2. **政治批判**：揭示国家、法律的阶级本质
3. **经济批判**：揭示资本主义生产方式的矛盾
4. **实践批判**：通过革命实践改变社会

小海豹问："这种批判和青年黑格尔派的批判有什么不同？"
我回答："青年黑格尔派只在**思想领域**批判，认为改变了思想就改变了世界。我们主张：要改变思想，必须先改变**产生思想的社会条件**。这是**实践批判**和**纯粹思想批判**的根本区别。"

## 词条20：马恩哲学的遗产——哲学的终结与新生

### 官方解释
马克思、恩格斯哲学标志着传统思辨哲学的终结和实践哲学的开端。哲学不再是无根的思辨，而是扎根于历史实践的批判武器。哲学的任务从解释世界转变为改变世界。

### 1840年兔狲老师解释
（我看着完成的《资本论》手稿）

"传统哲学**终结**了，但哲学获得了**新生**。"

**哲学的终结**：
1. **思辨哲学的终结**：脱离实践的纯粹思辨走到尽头
2. **体系哲学的终结**：试图建立包罗万象的哲学体系不再可能
3. **解释哲学的终结**：只为解释世界而存在的哲学失去意义

**哲学的新生**：
1. **实践哲学的开端**：哲学扎根于**物质生产实践**
2. **批判哲学的发展**：哲学成为**批判现实**的理论武器
3. **革命哲学的创立**：哲学为**社会革命**提供理论指导
4. **科学哲学的奠基**：哲学与**具体科学**（政治经济学、历史学）结合

**马恩哲学的核心贡献**：
1. **实践转向**：从思辨到实践，从解释到改变
2. **历史转向**：从永恒真理到历史具体，从抽象人性到历史的人
3. **阶级转向**：从普遍理性到阶级立场，从抽象批判到阶级分析
4. **总体转向**：从孤立范畴到有机整体，从部分分析到总体把握

**对后来哲学的影响**：
1. **西方马克思主义**：卢卡奇、葛兰西、法兰克福学派等发展批判理论
2. **结构主义马克思主义**：阿尔都塞等用结构主义方法重新解释
3. **后马克思主义**：拉克劳、墨菲等结合后结构主义
4. **解放哲学**：第三世界的解放神学、女权主义马克思主义等

**当代意义**：
1. **批判工具**：提供分析资本主义的批判工具
2. **解放视野**：指向人类解放的哲学视野
3. **实践智慧**：强调理论和实践统一的智慧
4. **历史意识**：培养历史具体性的思维方式

小小猪问："哲学终结了，那我们还要学哲学吗？"
我回答："要学的不是**终结的哲学**，而是**新生的哲学**——不是解释世界的装饰品，而是改变世界的批判武器。不是书斋里的思辨游戏，而是与实践结合的理论指南。"

### 思考题39：动手题

## 第四部分小结：实践哲学的创立

**马恩哲学的革命**：
1. **基础的革命**：从思辨到实践，从思想到物质生产
2. **方法的革命**：历史唯物主义、阶级分析、总体性方法
3. **功能的革命**：从解释世界到改变世界，从装饰品到武器
4. **立场的革命**：从资产阶级立场到无产阶级立场

**放下哲学的意义**：
1. **发现局限**：纯粹哲学无法解决根本问题
2. **转向现实**：从思想批判转向现实批判
3. **深入研究**：政治经济学揭示社会的深层结构
4. **实践结合**：理论为革命实践服务

**哲学的终结与新生**：
1. **终结的**：脱离实践的思辨哲学、体系哲学、解释哲学
2. **新生的**：实践哲学、批判哲学、革命哲学
3. **转变的**：从"哲学王"到"哲学战士"，从书斋到战场

**兔狲老师的过渡**：
"我和恩格斯完成了一场哲学革命：把哲学从天上拉回地上，从思想拉回实践，从解释拉回改变。

但哲学的故事没有结束。德国观念论分裂出的三条道路都在继续发展：存在主义深入个体生存，分析哲学澄清语言逻辑，而实践哲学在历史中检验和发展。

现在，让我们展望哲学的未来。在1840年这个转折点上，我们看到哲学正在分化、转型、寻找新方向。这就是第五部分：企望新哲学——哲学的三条未来道路。"
