# 第三部分：德国观念论及其分裂——哲学的三条道路

> **历史背景**：18世纪末-19世纪中叶，德国。法国大革命震撼欧洲，但德国资产阶级软弱，只能在思想领域进行革命。黑格尔体系达到德国古典哲学顶峰，但随后分裂为不同方向。

## 词条11：康德——批判哲学的奠基

### 官方解释
伊曼努尔·康德（1724-1804），德国古典哲学创始人，批判哲学创立者。主要著作：《纯粹理性批判》（1781）、《实践理性批判》（1788）、《判断力批判》（1790）。核心问题：知识如何可能？道德如何可能？

### 1840年兔狲老师解释
（在柯尼斯堡大学，康德每天准时散步）

"康德开始了哲学的'哥白尼革命'：不是认识符合对象，而是对象符合认识。"

**批判哲学的任务**：
1. **为理性划界**：理性能够认识什么？不能认识什么？
2. **为信仰留地**：限制知识，为信仰腾出空间
3. **为道德奠基**：建立普遍的道德法则

**三大批判**：
1. **《纯粹理性批判》**（理论理性）：
   - **先验感性论**：时空是感性直观的先天形式
   - **先验分析论**：范畴是知性的先天形式（因果、实体等）
   - **先验辩证论**：理性超越经验会产生二律背反（宇宙有限/无限等）
   - **物自体**（Ding an sich）：不可知的本体世界

2. **《实践理性批判》**（实践理性）：
   - **道德律令**（定言命令）："要只按照你同时能够愿意它成为一个普遍法则的那个准则去行动"
   - **自由意志**：人是目的，不是手段
   - **至善**：德福一致需要上帝和不朽灵魂的假设

3. **《判断力批判》**（审美和目的论判断）：
   - **审美判断**：无目的的合目的性
   - **目的论判断**：自然好像有目的

**历史意义**：
- **进步性**：系统批判理性，为科学和道德奠基
- **问题**：现象/物自体二元对立，割裂理论和实践
- **阶级性**：反映**德国资产阶级**的软弱性：向往革命但害怕实践

小小猪问："康德的'物自体'为什么有问题？"
我回答："因为它割裂了**现象**和**本质**，认为本质不可知。但本质不是躲在现象背后的神秘'物自体'，而是通过现象**显现**出来的。我们要在**实践**中认识本质。"

### 思考题21：动手题

康德提出的四组二律背反：
1. **世界在时间上有开端/无开端，在空间上有限/无限**
2. **一切复合物都由单纯部分构成/没有单纯部分，一切都是复合的**
3. **世界上有出于自由的原因/没有自由，一切都是自然**

3. 黑格尔如何批判康德的解决方式？（矛盾是事物发展的动力）
4. 实践观点如何解决这些矛盾？（在历史实践中解决理论矛盾）

## 词条12：费希特与谢林——主观与客观的观念论

### 官方解释
约翰·戈特利布·费希特（1762-1814）和弗里德里希·威廉·约瑟夫·谢林（1775-1854）是康德和黑格尔之间的德国观念论哲学家。费希特发展主观观念论，谢林发展客观观念论。

### 1840年兔狲老师解释
（耶拿大学，费希特正在激情演讲）

"费希特和谢林试图克服康德的二元论，但走向了不同的方向。"

**费希特的主观观念论**：
1. **绝对自我**（Das absolute Ich）：一切知识的最高原则
   - 自我设定自身（正题）
   - 自我设定非我（反题）
   - 自我与非我统一（合题）
2. **知识学**（Wissenschaftslehre）：从绝对自我推导出整个知识体系
3. **实践优先**：行动先于认识，意志创造世界
4. **历史意义**：强调主体能动性，但陷入主观唯心主义

**谢林的客观观念论**：
1. **同一哲学**：自然和精神是绝对同一体的两个表现
   - 自然：沉睡的精神
   - 精神：觉醒的自然
2. **绝对者**（Das Absolute）：超越主体客体的无差别点
3. **艺术哲学**：艺术直观能把握绝对同一
4. **后期转向**：从理性哲学转向启示哲学、神话哲学
5. **历史意义**：试图统一自然和精神，但带有神秘主义倾向

**共同问题**：
1. **唯心主义**：把思想当作第一性的存在
2. **抽象性**：脱离具体历史和社会条件
3. **体系封闭**：试图用逻辑推导出整个世界

小海豹问："他们和康德有什么不同？"
我回答："康德停在二元对立（现象/物自体），费希特用**绝对自我**统一，谢林用**绝对同一**统一。但都是在**思想领域**内统一，不是在实践中统一。"

|------|-------------|-----------------|---------------|

## 词条13：黑格尔——绝对观念论的顶峰

### 官方解释
格奥尔格·威廉·弗里德里希·黑格尔（1770-1831），德国古典哲学集大成者，绝对观念论创立者。主要著作：《精神现象学》（1807）、《逻辑学》（1812-1816）、《法哲学原理》（1821）。建立庞大的辩证体系：逻辑学、自然哲学、精神哲学。

### 1840年兔狲老师解释
（柏林大学，黑格尔正在讲授他的体系）

"黑格尔建立了哲学史上最庞大的唯心主义体系，但体系内部有革命性的辩证法。"

**黑格尔体系的结构**：
1. **逻辑学**：纯粹概念的运动（存在论、本质论、概念论）
2. **自然哲学**：理念的外化（力学、物理学、有机物理学）
3. **精神哲学**：理念的回归
   - 主观精神（人类学、现象学、心理学）
   - 客观精神（法、道德、伦理：家庭、市民社会、国家）
   - 绝对精神（艺术、宗教、哲学）

**核心概念**：
1. **绝对精神**（Der absolute Geist）：宇宙的理性本质，通过历史实现自身
2. **辩证法**（Dialektik）：
   - **正题-反题-合题**（These-Antithese-Synthese）
   - **否定之否定**：通过否定达到更高肯定
   - **矛盾推动发展**：事物在内在矛盾中发展
3. **历史主义**：理性在历史中展开，历史是理性的实现过程
4. **具体普遍**：普遍不是抽象的，而是在特殊中实现的

**著名命题**：
1. "凡是现实的都是合理的，凡是合理的都是现实的"
   - 表面：为现状辩护
   - 深层：现实会变化，不合理的会变成不合理
2. "密涅瓦的猫头鹰在黄昏起飞"：哲学反思总是在事件之后

**历史评价**：
- **贡献**：辩证法、历史主义、系统思维、矛盾发展观
- **问题**：唯心主义（绝对精神）、体系封闭、与现实妥协
- **阶级性**：反映**德国资产阶级**的矛盾：向往革命但害怕群众运动

恩格斯对我说："黑格尔的体系是**头足倒置**的：不是思想决定存在，而是存在决定思想。我们要把辩证法'倒过来'，让它用脚站立。"

### 思考题25：动手题
问题：分析黑格尔辩证法三定律：

黑格尔辩证法（被后人总结为三定律）：
1. **质量互变规律**：量变积累引起质变
   - 例子：水加热到100°C变成蒸汽
   - 社会例子：革命条件的成熟

2. **对立统一规律**：矛盾双方既对立又统一，推动发展
   - 例子：磁铁的南北极
   - 社会例子：阶级矛盾推动社会发展

3. **否定之否定规律**：肯定→否定→否定之否定（更高肯定）
   - 例子：种子→植物→新种子
   - 社会例子：原始公有制→私有制→共产主义公有制

从1840年视角分析：
1. 每个规律的合理内核是什么？
2. 在黑格尔体系中如何被唯心主义表述？
3. 如何"倒过来"变成唯物主义辩证法？
4. 如何用这些规律分析社会历史？

## 词条14：德国观念论的分裂——三条哲学道路

### 官方解释
黑格尔死后（1831年），他的学派分裂为老年黑格尔派（右派）和青年黑格尔派（左派）。随后进一步分裂为不同哲学方向：费尔巴哈的人本学唯物主义、马克思的历史唯物主义、克尔凯郭尔的存在主义等。

### 1840年兔狲老师解释
（柏林，黑格尔学派正在激烈争论）

"黑格尔体系像一座宏伟的大厦，但地基有问题——唯心主义。大厦必然分裂。"

**分裂的背景**：
1. **社会矛盾**：19世纪30-40年代，德国资本主义发展，阶级矛盾尖锐
2. **政治压迫**：普鲁士政府加强反动统治，压制自由思想
3. **理论危机**：黑格尔体系无法解释现实社会问题
4. **实践需要**：需要能指导改变世界的理论

**三条分裂道路**：

**第一条道路：向人的回归（费尔巴哈-马克思）**
1. **费尔巴哈**（1804-1872）：人本学唯物主义
   - 批判：把黑格尔"倒过来"——不是思想决定存在，而是存在决定思想
   - 贡献：恢复唯物主义的权威，批判宗教是人本质的异化
   - 局限：直观唯物主义，脱离实践和历史
   - 名言："人是人的最高本质"

2. **马克思**（1818-1883）：从哲学批判到政治经济学批判
   - 发现：形而上学的漏洞无法在思想领域缝合
   - 转向：从纯粹哲学批判转向**历史**和**政治经济学**研究
   - 核心：**实践**——改变世界的物质活动
   - 成果：历史唯物主义、剩余价值理论、科学社会主义

**第二条道路：向个人的回归（克尔凯郭尔-存在主义）**
1. **克尔凯郭尔**（1813-1855）：存在哲学先驱
   - 批判：黑格尔体系压抑**个体**、**选择**、**焦虑**
   - 转向：从普遍理性转向**个人存在**
   - 主题：恐惧、颤栗、信仰跳跃、生存三阶段（审美、伦理、宗教）
   - 影响：为20世纪存在主义奠基

2. **发展**：尼采、海德格尔、萨特等
   - 共同点：关注个体生存、自由选择、意义创造
   - 问题：容易陷入主观主义、非理性主义

**第三条道路：向语言的回归（分析哲学）**
1. **起源**：弗雷格、罗素、维特根斯坦等
   - 批判：传统哲学的"语言误用"
   - 转向：从形而上学问题转向**语言分析**
   - 方法：逻辑分析、概念澄清、意义理论
   - 目标：使哲学科学化、清晰化

2. **发展**：逻辑实证主义、日常语言学派、分析传统
   - 贡献：澄清概念、严格论证
   - 问题：脱离历史实践、技术化倾向

小小猪问："这三条道路有什么共同点？"
我回答："都是对黑格尔**绝对观念论**的反叛，都试图解决传统哲学的问题，但选择了不同的方向。第一条道路最彻底：不仅批判哲学，而且改变产生哲学问题的**社会条件**。"

## 词条15：马克思的转向——从哲学到政治经济学

### 官方解释
卡尔·马克思（1818-1883）早期是青年黑格尔派，受费尔巴哈影响转向唯物主义。1840年代发现哲学批判的局限性，转向政治经济学和历史研究，创立历史唯物主义和剩余价值理论。

### 1840年兔狲老师解释
（我在大英博物馆的阅览室，马克思正在研究经济资料）

"这是关键的时刻：哲学必须改变，否则就会死亡。"

**马克思的思想发展**：
1. **博士论文时期**（1839-1841）：研究德谟克利特和伊壁鸠鲁，强调自我意识
2. **《莱茵报》时期**（1842-1843）：接触现实问题（林木盗窃法、摩泽尔农民贫困）
3. **批判黑格尔法哲学**（1843）："不是国家决定市民社会，而是市民社会决定国家"
4. **《1844年经济学哲学手稿》**：异化劳动理论，受费尔巴哈影响
5. **《关于费尔巴哈的提纲》**（1845）："哲学家们只是用不同的方式解释世界，问题在于改变世界"
6. **《德意志意识形态》**（1845-1846）：与恩格斯合著，系统阐述历史唯物主义
7. **《哲学的贫困》**（1847）：批判蒲鲁东，进一步阐发历史唯物主义
8. **《共产党宣言》**（1848）："至今一切社会的历史都是阶级斗争的历史"

**关键转向**：
1. **从思想批判到现实批判**：
   - 发现：思想矛盾根源于**现实矛盾**
   - 转向：批判产生错误思想的**社会条件**

2. **从哲学到政治经济学**：
   - 发现：社会的基础是**物质生产**
   - 转向：研究资本主义**经济规律**

3. **从解释到改变**：
   - 发现：理论的价值在于指导**实践**
   - 转向：哲学要成为**无产阶级的精神武器**

**核心洞见**：
1. **实践观点**：人不是静观世界的认识主体，而是**改造世界的实践主体**
2. **历史观点**：思想是**历史的产物**，要在具体历史条件下理解
3. **阶级观点**：思想反映**阶级利益**，要分析思想的社会基础
4. **总体观点**：社会是**有机整体**，经济、政治、思想相互关联

小海豹问："为什么马克思要转向政治经济学？"
我回答："因为他在哲学批判中发现：思想的问题根源于**现实的问题**。要批判思想，必须批判产生思想的**现实**。而现实的核心是**经济关系**。所以必须研究政治经济学。"

## 第三部分小结：德国观念论的分裂与超越

**德国观念论的成就**：
1. **理性的自我批判**：康德开始，黑格尔完成的理性自我反思
2. **辩证思维的确立**：矛盾、发展、历史性的思维方式
3. **系统哲学的顶峰**：试图建立包罗万象的哲学体系
4. **历史意识的深化**：理性在历史中展开的历史主义

**德国观念论的问题**：
1. **唯心主义基础**：思想决定存在，头足倒置
2. **脱离现实实践**：在思想领域内循环
3. **体系封闭倾向**：历史终结，哲学完成
4. **压抑个体性**：普遍理性压抑个人选择

**分裂的三条道路**：
1. **向人的回归**（费尔巴哈-马克思）：恢复唯物主义，转向实践和历史
2. **向个人的回归**（克尔凯郭尔-存在主义）：关注个体生存、自由选择
3. **向语言的回归**（分析哲学）：语言分析，概念澄清

**马克思转向的革命意义**：
1. **哲学基础的变革**：从思辨到实践，从解释到改变
2. **研究对象的变革**：从纯粹思想到社会现实，特别是经济关系
3. **方法的变革**：历史唯物主义，阶级分析，总体性方法
4. **目的的变革**：哲学成为改变世界的理论武器

**兔狲老师的总结**：
"德国观念论完成了哲学在**思想领域**内的自我批判和自我发展，达到了唯心主义哲学的顶峰。但顶峰也是终点：哲学不能在纯粹思想领域内解决自己的问题。

黑格尔体系的**分裂**是必然的。不同的哲学家选择了不同的突围方向：有的回到**个人存在**，有的转向**语言分析**，有的深入**社会现实**。

我和恩格斯选择了第三条道路：不仅批判哲学，而且批判产生哲学问题的**社会条件**；不仅解释世界，而且**改变世界**；不仅思考思想，而且思考思想的**物质基础**。

这是哲学的**实践转向**，也是哲学的**新生**。哲学不再是无根的思辨，而是扎根于**历史实践**的批判武器。

现在，让我们进入第四部分：马恩哲学的诞生——实践哲学的创立。看看哲学如何从解释世界的工具，变成改变世界的武器。"
