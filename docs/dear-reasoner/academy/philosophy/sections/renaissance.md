# 第二部分：文艺复兴与启蒙思想——人文主义与科学革命

> **历史背景**：14-18世纪，欧洲从中世纪向现代社会过渡。资本主义萌芽、地理大发现、科学革命、宗教改革、资产阶级革命等重大变革。

## 词条6：文艺复兴——人的重新发现

### 官方解释
文艺复兴（14-17世纪）是欧洲从中世纪向近代过渡的文化运动，起源于意大利，后扩展到全欧洲。核心思想：人文主义（Humanism），强调人的价值、尊严和理性，复兴古希腊罗马文化。

### 1840年兔狲老师解释
（我站在佛罗伦萨的乌菲兹美术馆前，看着达芬奇的画作）

"文艺复兴不是简单的'复古'，而是**新兴资产阶级**的文化革命。"

**历史条件**：
1. **经济基础**：意大利城邦的**商业资本主义**发展，美第奇家族等银行家资助艺术
2. **政治变化**：封建贵族衰落，城市共和国兴起
3. **技术条件**：印刷术传入欧洲，知识传播加速
4. **外部刺激**：奥斯曼帝国攻占君士坦丁堡，希腊学者西逃带来古典文献

**核心思想：人文主义**：
1. **人的价值**：从中世纪的"神本"转向"人本"
2. **现世幸福**：追求现世成就，不是来世救赎
3. **理性尊严**：人凭借理性可以认识世界、创造文明
4. **个性解放**：反对禁欲主义，肯定人的情感和欲望

**代表人物**：
- **彼特拉克**：人文主义之父，强调古典研究与现实关怀
- **达芬奇**：通才（艺术家、科学家、工程师），体现人的无限潜能
- **马基雅维利**：《君主论》，政治现实主义，脱离道德说教

**历史意义**：
- **进步性**：打破神学束缚，解放人的思想创造力
- **阶级性**：主要是**资产阶级**和**城市贵族**的意识形态
- **局限性**：仍保留宗教框架，主要影响精英阶层

小小猪问："文艺复兴和古希腊有什么不同？"
我回答："古希腊的理性是**奴隶主**的理性，文艺复兴的理性是**资产阶级**的理性。前者为奴隶制辩护，后者为资本主义开路。"

## 词条7：科学革命——新世界观的诞生

### 官方解释
科学革命（16-17世纪）是近代科学的形成时期，代表人物：哥白尼、伽利略、开普勒、牛顿。核心成就：日心说、实验方法、数学化自然、牛顿力学体系。

### 1840年兔狲老师解释
（我翻开牛顿的《自然哲学的数学原理》）

"科学革命不仅是知识进步，更是**世界观**的革命。"

**哥白尼革命**（1543年《天体运行论》）：
- 内容：日心说代替地心说
- 意义：地球不是宇宙中心，人类不是特殊存在
- 影响：动摇神学世界观，开启科学自主

**伽利略的方法革命**：
1. **实验方法**：不是思辨，而是控制条件下的观察
2. **数学语言**：用数学描述自然规律
3. **望远镜观察**：扩展感官，发现新事实（木星卫星、月球环形山）

**牛顿的综合**（1687年《原理》）：
- **万有引力定律**：统一天上和地上的运动
- **三大运动定律**：惯性、加速度、作用反作用
- **世界观**：宇宙是机械的、可预测的、数学化的

**哲学意义**：
1. **自然观的变革**：自然不是神秘意志，而是遵循规律的机器
2. **认识论的变革**：知识来自观察和数学推理，不是权威和启示
3. **方法论的变革**：实验+数学+推理的科学方法确立

**历史唯物主义分析**：
1. **物质基础**：航海、采矿、军事等**实践需要**推动科学发展
2. **阶级动力**：**资产阶级**需要实用知识发展生产、贸易
3. **意识形态**：机械自然观为**机械唯物主义**哲学奠基
4. **历史局限**：仍保留"第一推动力"等神学残余

小海豹问："科学革命和哲学有什么关系？"
我回答："科学革命产生了**机械唯物主义**哲学：世界是物质的机器，人也是机器。这为后来的唯物主义发展奠定了基础，但也有局限性：把世界看作静态的、机械的。"

## 词条8：启蒙运动——理性的权威

### 官方解释
启蒙运动（17-18世纪）是欧洲的思想解放运动，核心口号："勇于运用你自己的理性！"（康德）。代表人物：伏尔泰、孟德斯鸠、卢梭、狄德罗、休谟、康德。强调理性、自由、进步、宽容。

### 1840年兔狲老师解释
（巴黎的咖啡馆里，启蒙思想家正在激烈辩论）

"启蒙运动是**资产阶级**的意识形态总动员，为政治革命做思想准备。"

**核心观念**：
1. **理性至上**：用理性检验一切——宗教、政治、道德、传统
2. **自由平等**：人生而自由平等，反对封建特权
3. **进步信念**：历史是进步的过程，理性推动社会完善
4. **自然权利**：人拥有不可剥夺的自然权利（生命、自由、财产）
5. **社会契约**：政府权力来自被统治者的同意

**主要流派**：
1. **法国启蒙**（伏尔泰、孟德斯鸠、卢梭、狄德罗）：
   - 特点：激进批判，直接为革命造舆论
   - 贡献：百科全书派传播知识，三权分立理论，社会契约论
   - 局限：抽象理性，脱离具体历史条件

2. **苏格兰启蒙**（休谟、亚当·斯密）：
   - 特点：经验主义、保守渐进
   - 贡献：怀疑论批判，政治经济学创立
   - 局限：为资本主义秩序辩护

3. **德国启蒙**（莱辛、康德）：
   - 特点：调和理性与信仰，强调道德自律
   - 贡献：批判哲学，永久和平理念
   - 局限：脱离政治实践，停留在思想领域

**历史评价**：
- **进步性**：批判封建专制和宗教迷信，为资产阶级革命提供理论武器
- **阶级性**：**资产阶级**的意识形态，反映工商业阶级利益
- **抽象性**：建立在**抽象人性**、**抽象理性**基础上
- **矛盾性**：宣扬"普遍人权"，但实践中排除妇女、穷人、殖民地人民

恩格斯对我说："启蒙学者的'理性王国'不过是**资产阶级的理想化王国**。自由是贸易自由，平等是法律面前的资产阶级平等，所有权是资产阶级所有权。"

## 词条9：经验主义与理性主义——认识论的争论

### 官方解释
17-18世纪哲学围绕认识论展开激烈争论：经验主义（英国：洛克、贝克莱、休谟）认为知识来自感官经验；理性主义（大陆：笛卡尔、斯宾诺莎、莱布尼茨）认为知识来自理性推理和先天观念。

### 1840年兔狲老师解释
（我在大英博物馆的阅览室，对比洛克和笛卡尔的著作）

"经验主义和理性主义的争论，是**资产阶级哲学**在认识论上的内部分歧。"

**经验主义**（英国传统）：
- **洛克**（1632-1704）：白板说（tabula rasa），知识来自经验
  - 进步：反对先天观念，为科学经验方法奠基
  - 局限：被动反映论，忽视主体能动性
  - 政治：为1688年光荣革命辩护，主张有限政府

- **贝克莱**（1685-1753）：主观唯心主义，"存在就是被感知"
  - 问题：否认物质客观存在，为宗教服务
  - 启示：暴露经验主义的唯心主义倾向

- **休谟**（1711-1776）：彻底经验主义，怀疑论
  - 贡献：批判因果关系的必然性，揭示经验主义困境
  - 问题：导致不可知论，动摇科学基础

**理性主义**（大陆传统）：
- **笛卡尔**（1596-1650）：理性主义奠基人
  - 方法：普遍怀疑，"我思故我在"
  - 贡献：强调理性自主，为科学提供方法论
  - 问题：身心二元论，脱离经验

- **斯宾诺莎**（1632-1677）：唯物主义倾向的理性主义
  - 理论：实体一元论（神即自然），决定论
  - 贡献：批判宗教，接近唯物主义
  - 局限：几何学方法，脱离历史具体

- **莱布尼茨**（1646-1716）：客观唯心主义
  - 理论：单子论，预定和谐
  - 问题：神秘主义，为现状辩护

**共同问题**：
1. **抽象主体**：假设孤立的、抽象的"认识主体"
2. **脱离实践**：在纯粹思想领域讨论认识问题
3. **非历史性**：不考虑认识的历史发展和社会条件

小小猪问："那正确的认识论应该是什么？"
我回答："认识是在**社会实践**中产生的。人不是静观世界的旁观者，而是**改造世界的行动者**。认识的主体不是抽象的个人，而是**社会的人**，在具体历史条件下从事实践活动。"

## 词条10：文艺复兴与启蒙思想的总结

### 官方解释
文艺复兴和启蒙运动是欧洲从中世纪向现代过渡的关键时期，确立了理性、自由、科学、进步等现代价值观念，为现代社会的形成奠定了思想基础。

### 1840年兔狲老师解释
（我合上启蒙运动的著作，看着窗外蒸汽机冒出的浓烟）

"文艺复兴和启蒙运动完成了**资产阶级的思想解放**，但解放的是**资产阶级**，不是全人类。"

**历史贡献**：
1. **理性权威的确立**：用理性代替信仰作为最高权威
2. **人的重新发现**：肯定人的价值、尊严和创造力
3. **科学方法的形成**：实验、观察、数学推理的科学传统
4. **自由平等观念的传播**：为民主革命提供理论武器
5. **进步历史观的确立**：历史是理性推动的进步过程

**历史局限**：
1. **阶级局限性**：主要是**资产阶级**的意识形态
   - 人文主义的"人"是资产阶级个人
   - 启蒙的"理性"是资产阶级理性
   - 自由的"自由"是贸易自由和财产自由

2. **抽象性缺陷**：
   - 抽象人性：假设永恒不变的"人性"
   - 抽象理性：脱离具体历史条件的"纯粹理性"
   - 抽象权利：脱离经济基础的"自然权利"

3. **意识形态功能**：
   - 为资本主义秩序辩护
   - 掩盖阶级矛盾和剥削
   - 把特定历史形式普遍化、永恒化

4. **实践矛盾**：
   - 宣扬"普遍人权"，但实践中排除大多数人
   - 倡导"自由平等"，但维护财产不平等
   - 相信"理性进步"，但产生新的压迫形式

**1840年批判视角**：
1. **历史具体化**：把抽象观念放回具体历史条件
   - 不是"人性"永恒，而是**历史的人**
   - 不是"理性"纯粹，而是**阶级的理性**
   - 不是"权利"自然，而是**社会的权利**

2. **阶级分析**：揭示思想的社会基础
   - 谁的思想？为谁服务？
   - 反映什么阶级利益？
   - 掩盖什么社会矛盾？

3. **实践检验**：思想的价值在于指导改变世界的实践
   - 不是解释世界，而是改变世界
   - 理论要掌握群众，变成物质力量
   - 哲学要成为无产阶级的精神武器

小海豹沉思："所以我们要继承启蒙的批判精神，但要超越它的阶级局限？"
我点头："是的。启蒙运动教导我们**勇于运用理性**，我们要把这个教导推向彻底：勇于运用理性批判理性自身的历史条件，勇于运用理性改变产生非理性的社会条件。"

### 思考题19：动手题
问题：用历史唯物主义分析一个启蒙观念：

选择：**卢梭的"社会契约"**、**伏尔泰的"宗教宽容"**、**孟德斯鸠的"三权分立"**、**康德的"永久和平"**

分析框架：
1. **历史条件**：产生这个观念的具体历史条件（经济、政治、社会）
2. **阶级基础**：反映哪个阶级的利益和视角？
3. **理论贡献**：相对于封建思想有什么进步？
4. **历史局限**：受历史条件的什么限制？

6. **当代意义**：今天还有什么价值？需要如何改造？

## 第二部分小结：文艺复兴与启蒙的遗产

**核心成就**：
1. **理性传统的复兴与发展**：从中世纪信仰权威到现代理性权威
2. **人的主体地位的确立**：人文主义对人的价值、尊严的肯定
3. **科学世界观的形成**：机械自然观、实验方法、数学化
4. **现代政治理念的奠基**：自由、平等、民主、人权、社会契约
5. **进步历史观的确立**：历史是理性推动的进步过程

**历史局限**：
1. **阶级局限**：主要是**资产阶级**的意识形态
2. **抽象缺陷**：抽象人性、抽象理性、抽象权利
3. **实践矛盾**：理想与现实的差距，普遍宣称与特殊实践
4. **非历史性**：把特定历史形式普遍化、永恒化

**1840年批判的启示**：
1. **历史具体性原则**：思想要在具体历史条件下理解
2. **阶级分析原则**：揭示思想的社会基础和阶级立场
3. **实践检验原则**：思想的价值在于指导改变世界的实践
4. **辩证继承原则**：在批判中继承，在继承中超越

**兔狲老师的过渡**：
"文艺复兴和启蒙运动把理性从神学束缚中解放出来，但理性又被**资产阶级**关进了新的牢笼：抽象性、形式化、为现状辩护。

现在，理性需要新的解放。这个任务落在了**德国哲学**身上——特别是康德、费希特、谢林、黑格尔的德国观念论。他们试图在思想领域内完成理性的自我批判和自我发展。

但思想领域的革命终究是有限的。真正的理性解放需要在**实践**中完成。这就是我和恩格斯正在思考的问题：如何把哲学从解释世界的工具，变成改变世界的武器。

让我们进入第三部分：德国观念论——思想领域内的哲学革命。"
